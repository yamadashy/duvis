# duvis 📊

> **This npm package is a placeholder.** The full npm distribution is in development.

`duvis` (pronounced `/ˈduːvɪs/`, like "doo-vis") is a fast, read-only disk usage analyzer that helps both AI agents and humans understand what's filling their disk.

## Install (today)

For now, install from [crates.io](https://crates.io/crates/duvis):

```sh
cargo install duvis
```

## Install (coming soon)

```sh
npx duvis ~/projects
# or
npm install -g duvis
```

The npm distribution will ship prebuilt binaries per platform — no Rust toolchain required.
